# Alazab BIM — Project Information Management

منصة **Alazab BIM** هي مركز إدارة معلومات المشروع داخل العزب. الهدف ليس عرض ملفات فقط، بل تحويل كل مشروع إلى مرجع رقمي منظم: ملفات، اعتمادات، ملاحظات تنفيذ، RFI، Submittals، تقارير فنية، تسليم، وتشغيل بعد التسليم.

## خريطة الريبوهات داخل العزب

| Repository | الدور |
|---|---|
| `AlazabDev/bim-alazab` | إدارة BIM ومعلومات المشروع |
| `AlazabDev/az-studio` | إدارة التصميم والعناصر البارامترية |
| `AlazabDev/alazab-nexus` | إدارة المنتجات والكتالوجات التجارية |

## الوحدات الأساسية

1. **Projects** — كود المشروع، العميل، الفرع، الموقع، الحالة، ونسبة التقدم.
2. **Files & Revisions** — ملفات المشروع مع التصنيف، الإصدار، الرافع، والحالة.
3. **Approvals** — دورة الاعتماد: Draft, Submitted, Under Review, Approved, Rejected.
4. **Issues** — ملاحظات التنفيذ والتعارضات ومتابعة الإغلاق.
5. **RFI** — طلبات الاستفسار الفني وربطها بالملفات والمناطق.
6. **Submittals** — عينات، كتالوجات، Datasheets، وموافقات الموردين.
7. **Technical Evidence** — تقارير HVAC, Lighting, Plumbing وما يرتبط بها من حسابات.
8. **Handover** — As-Built، صور نهائية، ضمانات، O&M، وسجل التسليم.
9. **Post-Handover FM** — الصيانة، التشغيل، التعديلات المستقبلية، والرجوع للمعلومات.
