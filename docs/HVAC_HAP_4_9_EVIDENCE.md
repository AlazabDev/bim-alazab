# HVAC Evidence — Carrier HAP 4.9

## Status

Carrier HAP 4.9 has been installed, activated, and started successfully.

## Role inside Alazab BIM

HAP 4.9 is now the primary HVAC calculation evidence source for early load calculation and HVAC technical reports.

## Evidence workflow

1. Create project in HAP 4.9.
2. Define weather/location data.
3. Define spaces and zones.
4. Enter areas, occupancy, lighting load, equipment load, schedules, and ventilation assumptions.
5. Run cooling/heating load calculation.
6. Export report as PDF.
7. Upload report to Alazab BIM under Technical Evidence / HVAC.
8. Link the report to project code, revision, reviewer, and approval status.

## Minimum required fields

- Project code
- Branch or site name
- Prepared by
- Reviewed by
- HAP version: 4.9
- Calculation date
- Weather data source
- Design assumptions
- Spaces/zones list
- Cooling load summary
- Heating load summary if applicable
- Ventilation summary
- Equipment schedule if applicable
- Exported PDF report

## First validation test

Use one small Abu Auf branch or sample retail unit and generate the first HVAC evidence record.

The first report is not for final engineering approval. It is for validating the Alazab BIM evidence workflow end-to-end.
